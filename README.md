# System clinic CRUD API REST


## Technologies used

* Java 17
* Spring Boot 
* MySQL

## Functions 

### Employee
  * Create Employee
  * List All Employees
  * List Employee
  * Delete Employee

### Patient
  * Create Patient
  * List All Patients
  * List Patient
  * Delete Patient



  ## Rests da API

  ### Employee Controller

|Rest |Função |

|GET   | Consult All Employees |
